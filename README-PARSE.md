# بازیابی تمام سریال‌ها

برای استفاده از **تمام ۹۷۰+ سریال** با دسته‌بندی ژانری:

## مراحل

1. **فایل خام سریال‌ها را بسازید**
   - اگر نسخه قبلی `index.html` را دارید (با محتوای تمام سریال‌ها)، آن را باز کنید
   - قسمت محتوای سریال‌ها را کپی کنید (از اولین `<h3>1.` یا `<h3>7.` تا آخر)
   - یک فایل جدید به نام `series-raw.html` بسازید
   - محتوای کپی شده را در آن قرار دهید و ذخیره کنید

2. **اجرای اسکریپت**
   ```bash
   node parse-series.js
   ```

3. **خروجی**
   - فایل `series-data.js` با تمام سریال‌های استخراج‌شده و دسته‌بندی ژانری به‌روز می‌شود

## ژانرها

- دراما
- کمدی  
- جنایی
- علمی-تخیلی
- فانتزی
- اکشن
- ترسناک
- انیمه
- سایر

سایت در حال حاضر با ۶ سریال نمونه کار می‌کند. پس از اجرای parser با فایل خام، تمام سریال‌ها نمایش داده می‌شوند.
